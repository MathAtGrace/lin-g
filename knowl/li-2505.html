Item d.

Applying linearity, we see that

  1. \(A(4\vvec_1) = 4A\vvec_1 = 12\vvec_1\text{.}\)

  2. \(A(\vvec_1+\vvec_2) = A\vvec_1 + A\vvec_2 = 3\vvec_1 - \vvec_2\text{.}\)

  3. \(A(4\vvec_1-3\vvec_2) = 4A\vvec_1 -3A\vvec_2 = 12\vvec_1 + 3\vvec_2\text{.}\)

  4. \(A^2\vvec_1 = A(3\vvec_1) = 9\vvec_1\text{.}\)

  5. \(A^2(4\vvec_1-3\vvec_2) = A(12\vvec_1 +3\vvec_2) = 36\vvec_1 - 3\vvec_2\text{.}\)

  6. \(A^4\vvec_1 = 3^4\vvec_1 = 81\vvec_1\text{.}\)
